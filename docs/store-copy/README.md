# Store Copy Localization Structure

This folder is a lightweight source-of-truth scaffold for localized App Store copy.

## Files

- `en.md`
- `tr.md`
- `de.md`
- `es.md`
- `ja.md`
- `pt-BR.md`

Each file should mirror the same section order:

1. App Name
2. Subtitle
3. Promotional Text
4. Short Description
5. Long Description
6. Keywords

Keep this structure simple and editorial-first. Do not add runtime dependencies or app code integration for store copy.
