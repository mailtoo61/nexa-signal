# Real Asset Capture Prep Guide

## Screenshot Session Preparation

- Confirm target RC build label.
- Confirm intended locale before each capture pass.
- Confirm required device classes and orientation plan.

## Preview Video Session Preparation

- Prepare shotlist from approved sequence.
- Confirm capture path uses real gameplay only.
- Rehearse transitions to reduce retakes.

## Reduced-Motion Capture

- Capture at least one settings frame with reduced-motion enabled.
- Capture one representative gameplay moment with reduced-motion enabled.

## Localization Capture

- Capture base English first.
- Repeat same sequence per active locale.
- Validate locale-specific text and truncation before export.

## Clean-Device Capture Requirements

- Use clean install or clean state where possible.
- Clear stale notifications and unrelated overlays.
- Ensure no personal data appears on capture surface.

## Notification Suppression

- Enable focus/do-not-disturb mode.
- Disable pop-up notifications that can contaminate captures.

## Battery / Performance Prep

- Charge device sufficiently before capture.
- Avoid thermal-throttled sessions for final assets.
- Re-capture scenes if visible stutter appears.

## Visual Cleanliness Checklist

- No clipping in safe areas.
- Consistent brightness and contrast.
- Consistent visual mode strategy per asset set.

## No DEV/Debug Verification

- Confirm no development labels, panels, or tuning overlays are visible.
- Re-run production guard checks before final export sign-off.

## Export Organization Guidance

- Use deterministic file naming and folder structure per locale/device class.
- Keep versioned export batches for traceability.
