# App Store Screenshot Plan

## Objective

Present NEXA SIGNAL as a premium, calm, intelligent strategy title with clean UX and reliable session recovery.

## Target Device Groups

- iPhone 6.9" class (primary marketing set)
- iPhone 6.5" class (secondary)
- iPhone 5.5" class (compatibility set if required by current App Store rules)

Capture with production-like build configuration and production-visible UI only.

## Global Capture Rules

- No dev/debug/playtest overlays.
- No placeholder legal/support copy visible.
- Use consistent visual theme and typography.
- Ensure text legibility at thumbnail size.
- Keep overlays minimal; prioritize in-app UI authenticity.

## Recommended Screenshot Sequence (6-8 Frames)

1. Home Hero

- Screen: Home
- Focus: premium mood, clean entry point
- Suggested caption: `Guide a living signal network`

2. Core Decision Loop

- Screen: active gameplay with actionable node/link state
- Focus: strategic choices, readable tension
- Suggested caption: `Stabilize, repair, and reroute`

3. Mid-Run Pressure

- Screen: elevated risk moment before collapse
- Focus: urgency without chaos
- Suggested caption: `Every move affects the whole grid`

4. Collapse & Recovery Context

- Screen: session summary / collapse state
- Focus: consequence and clarity
- Suggested caption: `Learn from each collapse`

5. Resume Reliability

- Screen: Home with Continue Network available
- Focus: deterministic resume value
- Suggested caption: `Resume cleanly, continue precisely`

6. Settings & Accessibility

- Screen: Settings panel (audio, haptics, reduced motion, language)
- Focus: player control and polish
- Suggested caption: `Tune feedback to your play style`

7. Language / Localization Proof (optional but recommended)

- Screen: same UI in secondary locale (e.g., Turkish)
- Focus: localization readiness
- Suggested caption: `Prepared for global release`

8. Reduced Motion Proof (optional)

- Screen: Settings reduced motion enabled + calm gameplay frame
- Focus: accessible motion control
- Suggested caption: `Reduced motion when you prefer it`

## Home / Gameplay / Recovery Coverage Checklist

- Home entry state
- Continue state (validated resume present)
- Standard gameplay action frame
- High-risk gameplay frame
- Collapse or session-end summary frame
- Settings accessibility frame

## Optional App Preview Video Shot List (15-30s)

1. Home -> Enter Network (2-3s)
2. Tactical action sequence: stabilize/repair/connect (6-8s)
3. Pressure escalation moment (3-4s)
4. Session end + immediate restart/continue context (3-4s)
5. Settings quick pass (audio/haptics/reduced motion/language) (3-4s)
6. Closing brand frame (2s)

Use subtle transitions; avoid aggressive motion or rapid cuts.

## Localization Screenshot Notes

- Prioritize English for initial submission unless localized storefront launch is simultaneous.
- If localizing screenshots, ensure caption text and in-app text language match exactly.
- Re-check truncation in long German/Turkish labels before export.
