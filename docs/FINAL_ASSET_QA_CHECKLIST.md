# Final Asset QA Checklist

## Screenshots

- Primary iPhone class screenshots complete
- Secondary/required classes complete
- Gameplay states are real and representative
- No placeholder strings visible

## Localized Screenshots

- Each launch locale set captured
- Locale text matches storefront metadata language
- No truncation in critical labels

## Preview Video

- Uses real gameplay only
- Includes Home, gameplay, recovery, settings moments
- No dev/debug overlays

## App Icon / Splash Assets

- Final icon set exported and verified
- Splash assets match brand and current build visuals

## App Store Captions

- Captions align with screenshot order and tone
- No exaggerated or unverified claims

## Export Dimensions / Compression

- Dimensions match App Store requirements
- Compression quality remains crisp at upload scale

## Safe-Area Checks

- Key UI elements not clipped in captured assets
- Text remains readable in all target device classes

## Dark/Light Consistency

- Asset set uses consistent mode strategy per locale set
- Mixed-mode usage is intentional and documented

## Production Safety

- No DEV/debug/playtest labels in assets
- No unreleased/future feature UI surfaced unintentionally
