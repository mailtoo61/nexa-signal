# App Store Connect Ready State Guide

## App Record Readiness

- App record exists and matches final bundle id.
- Platform/category fields are complete.

## Metadata Readiness

- Core metadata fields complete for active submission locales.
- No placeholders in active fields.

## Localized Metadata Readiness

- Launch locales finalized.
- Non-launch locales scoped and clearly statused.

## Screenshot Readiness

- Required device-class sets exported and QA-verified.
- Locale and caption consistency validated.

## Preview Video Readiness

- Included only if final and representative.
- Verified against no-fake-gameplay requirement.

## Privacy / Support Readiness

- Privacy/terms/support paths reachable and current.
- Support path reflects operationally monitored contact.

## Reviewer Notes Readiness

- Notes are concise, factual, and actionable.
- Notes avoid unverified claims.

## Release Config Freeze Readiness

- Runtime config values locked for submission candidate.
- No placeholder release identifiers remain.

## Final Build Readiness

- Correct RC build selected.
- Label/version/build mapping verified.

## Submission Readiness

- Submission package checklist complete.
- Final authorization template ready for sign-off.
