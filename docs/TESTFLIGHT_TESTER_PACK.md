# TestFlight Tester Pack

## Welcome Message (Draft)

Thank you for helping test NEXA SIGNAL.
This build is focused on gameplay stability, deterministic resume/recovery behavior, and production-level polish.
Your feedback directly improves launch quality.

## What To Test

- First launch and entry flow
- Core gameplay loop
- Continue Network behavior
- Settings persistence (audio, haptics, reduced motion, language)
- Collapse/recovery transitions
- Legal/support links

## How To Report Bugs

- Use the reproduction template in `docs/BETA_FEEDBACK_WORKFLOW.md`
- Include:
  - device model
  - iOS version
  - build label
  - clear reproduction steps
  - screenshot or short clip when possible

## Expected Rough Edges

- Minor copy/formatting adjustments may still occur.
- Some localized store-facing content is still in editorial progress.

## Accessibility / Reduced-Motion Notes

- Please explicitly test reduced motion behavior.
- Report any readability or interaction concerns.
- Include whether VoiceOver/reduced motion were enabled.

## Known Limitations

- Core gameplay is local-first with no cloud sync.
- Some future-facing features are intentionally deferred.

## Support Expectations

- Reports are triaged by severity.
- Critical issues receive highest priority.
- You should receive acknowledgement within the beta response window defined by the release team.
