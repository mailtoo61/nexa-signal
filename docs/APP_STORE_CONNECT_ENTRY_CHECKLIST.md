# App Store Connect Entry Checklist

## Metadata Fields

- App name
- Subtitle
- Promotional text
- Description (short/long as applicable)
- Keywords

## Screenshot Uploads

- Primary iPhone class screenshot set uploaded
- Secondary class set uploaded if required
- No placeholder/dev overlays in uploaded media

## Localization Entries

- Active locales have complete metadata entries
- Locale copy reviewed for truncation and quality

## Support / Privacy URLs

- Privacy URL verified (HTTPS, reachable)
- Terms URL verified (HTTPS, reachable)
- Support URL or support email verified

## Export Compliance

- Export compliance prompts answered consistently and truthfully

## Review Notes

- Reviewer notes include concise testing guidance
- Notes mention core flow and settings access where helpful

## Age Rating

- Age rating questionnaire completed and reviewed

## Categories

- Primary/secondary categories confirmed

## Build Selection

- Correct RC build selected for submission
- Build label/version/build number cross-checked

## Phased Release Settings

- Phased release preference decided and documented

## Future Check: Game Center

- If Game Center is added later:
  - feature flags and metadata updated
  - review notes adjusted accordingly

## Accessibility Notes (Optional but Recommended)

- Mention reduced-motion and feedback controls in reviewer context if relevant
